Feature Overview

The Send a Hug feature allows partners to send virtual hugs to each other with:

  • Real-time Firebase notifications
  • Haptic feedback (vibration) on mobile devices
  • Beautiful floating heart animations
  • Browser notifications with custom messages
  • Toast notifications with action buttons
  • Daily hug counters (sent/received)

How to Test in App

  1. Build and run the app: npm run build:mobile
  2. Open on two devices with different user accounts
  3. Connect both users as a couple
  4. Tap the "Send a Hug" button on one device
  5. The other device should receive:
    • Haptic vibration
    • Toast notification
    • Browser notification (if permitted)
    • Updated hug counter
